# Проект "Mesto Russia" #

Проект "Mesto Russia" - это лендинг, созданный для закрепления навыков верстки и программирования.

**Технологии**

* HTML;
* CSS;
* JavaScript;
* Отзывчивая вёрстка;
* БЭМ методология;
* Webpack;

**Ссылка на проект**

https://nerevar123.github.io/mesto/index.html
